# PetSitterApp

A Blazor WebAssembly PWA for managing pet sitting appointments, customers, and payments.

## Features
*   **Offline Support**: Works without an internet connection using IndexedDB.
*   **Google Integration**: Syncs data to Google Sheets and appointments to Google Calendar.
*   **Responsive UI**: Built with MudBlazor.
